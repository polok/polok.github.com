Touch a Cloud With CloudBees

For a long time I wanted to learn more about the cloud. Finally I had some time to try this. To start with, I did some research to find a provider who provides a basic range of services just for free. I found quite many, e.g:
Redhat Openshift
CloudBees
Jelastic

I had a simple REST web serivice which has been written using Spring 3.1 and to store data I have chosen MySQL. Well, I needed:

Tomcat on which I can deploy my service
Database

I chose CloudBeese (pricing details - Pricing) Without any problems in few minutes I was able to successfully set up Tomcat 7.X and MySQL database.

My database properties looked as follows:

db.name=MY_DATABASE_NAME
db.driver=com.mysql.jdbc.Driver
db.url=jdbc:mysql://URL_FROM_CLOUDBEES:PORT_FROM_CLOUDBEES/
db.user=MY_USER_NAME
db.pass=MY_USER_PASSWORD

To be honest, I was much easier as I thought. In few days I will put my simple REST web service into github.
